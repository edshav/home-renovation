# Home Renovation Landing Page

Simple, fast, and mobile-first landing page built for advertising on platforms like Kijiji.

## 🚀 Overview

This project is a static one-page website designed to showcase home renovation services and convert visitors into calls.

- Clean HTML + CSS (no frameworks)
- Optimized for mobile devices
- Fast loading (minimal dependencies)
- Image-focused layout for better conversion

## 📸 Features

- Hero section with strong CTA (Call Now)
- Image gallery (10 vertical photos)
- Services list
- Benefits section
- Sticky bottom call button (mobile-friendly)
- Click-to-call support (`tel:` links)

## 🛠 Usage

1. Replace images in the `/images` folder
2. Update phone number and email in `index.html`
3. (Optional) Customize text content

## 🌐 Deployment

This project is intended to be deployed via GitHub Pages:

1. Push the repo to GitHub
2. Go to repository Settings → Pages
3. Select the `main` branch
4. Your site will be available at: https://<username>.github.io/<repo-name>/

## ⚡ Performance Tips

- Use compressed images (`.webp`, ~1200px width)
- Keep file sizes small for faster loading
- Avoid adding unnecessary scripts

## 📞 Goal

Drive users from ads to a simple page where they can quickly view work examples and call directly.
